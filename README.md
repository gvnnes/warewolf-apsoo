# Como rodar o programa localmente

Siga os passos abaixo para executar o programa em sua máquina local:

1. **Pré-requisitos**  
    Certifique-se de ter o [Node.js](https://nodejs.org/) instalado em sua máquina.

2. **Instalação das dependências**  
    No diretório do backend e do frontend, execute o comando abaixo para instalar todas as dependências necessárias:
    ```
    npm install
    ```

3. **Execução do programa**  
    Após a instalação das dependências,  para iniciar o backend, dentro de __warewolf-backend__ use o comando:
    
    ```
    node server
    ```
   
    
    E  para iniciar o frontend dentro de __warewolf-gerencia__ use o comando:

    
    ```
    npm run 
    ```

4. **Acessando a aplicação**  
    Normalmente, a aplicação estará disponível na porta especificada no terminal, basta apertar __ctrl + botão esquerdo__.

5. **Dúvidas ou problemas**  
    Caso encontre algum problema, verifique se todas as dependências estão instaladas corretamente e se o Node.js está atualizado.
